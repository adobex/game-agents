# Media Buying Pace Analysis

用于拆解竞品的买量节奏、渠道分布和节点放量逻辑。

## 何时使用

- 需要估算竞品在预热期、首发爆发期、稳定期的投放节奏
- 需要判断 Meta、Google、TikTok、SDK 等渠道的相对侧重
- 需要分析节日、版本更新、节点事件如何影响放量

## 核心关注点

- 投放是一次性爆量，还是渐进爬坡后节点爆破
- 预算分布是否与产品生命周期和节日节点匹配
- 不同渠道承担的是拉新、补量、回收，还是品牌扩散
- 放量节奏背后是模型学习、榜单冲刺，还是事件营销

## 输出建议

- 尽量用阶段表格表示节奏
- 渠道占比若无精确数据，标注“推测/区间/相对高低”
- 重点解释“为什么在那个时间点这么投”
