# ASO Analysis

用于拆解一个产品在商店面、小游戏入口面或落地页入口面的信息架构与转化表达。

## 何时使用

- 需要分析竞品的关键词布局、描述、Icon 演变
- 需要总结核心转化标签与截图 / 视频传达的卖点
- 需要判断它在商店首屏如何完成“用户停留 -> 兴趣 -> 点击”

## 核心关注点

- 关键词是否围绕高意图需求词或赛道词
- 商店描述是否突出真正拉转化的核心卖点
- Icon、截图、短视频是否在首屏就传达差异化
- 转化标签是功能利益、情感利益，还是身份/场景利益

## 输出建议

- 先给 3-5 个核心转化标签
- 再拆关键词、描述、Icon、截图策略
- 最后回答“它为什么这么摆”和“这会吸到什么用户”
