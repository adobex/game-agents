# Creative Strategy Analysis

用于分析竞品广告素材的创意套路、素材池结构和迭代方式。

## 何时使用

- 需要总结竞品 Top 3 素材打法
- 需要判断哪些素材负责吸量，哪些负责留存或付费转化
- 需要分析素材衰减后的迭代逻辑

## 核心关注点

- 前 3 秒钩子是什么
- 素材展示的是真玩法、伪玩法、剧情、真人、UGC 还是结果导向
- 高 CTR 素材与高留存 / 高付费素材如何分工
- 素材衰减后是换包装、换卖点、换人群，还是换节点场景

## 输出建议

- 先列 Top 3 套路
- 再写各自更偏向吸量 / 留存 / 转化哪一端
- 最后总结素材池如何组合使用，而不是孤立点评单条素材
