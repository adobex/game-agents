# Bidding Strategy Analysis

用于从公开信号反推一个产品的常见出价模式与投放优化逻辑。

## 何时使用

- 需要判断竞品更偏向 Lowest Cost、tCPA、tROAS 还是混合策略
- 需要分析不同阶段的优化事件与回收目标
- 需要比较不同国家 / 区域的投放差异

## 核心关注点

- 冷启动期更重安装、激活还是付费事件
- 稳定期是保规模、保 ROI，还是分计划混合
- 不同地区的素材、版位、优化目标是否不同
- 平台数据回传能力是否带来模型优势

## 输出建议

- 区分“推测事实”和“推测逻辑”
- 优先解释为什么这种产品会用这种出价方式
- 分地区差异尽量对应用户画像、支付能力、平台结构和素材适配
