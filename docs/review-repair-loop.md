# Review Repair Loop

用于定义 `game-agents` 中“审查 -> 修复 -> 选择性重审 -> 终止”的闭环规则。它不新建一套平行组织，而是把参考2中最有价值的闭环机制映射到现有三省六部体系：

- `shangshu`：负责任务回收、修订派发、轮次追踪
- `menxia`：负责汇总裁决、是否重审、何时终止
- 对应 11 司：负责定点回修，不“顺手重写整案”

## 1. 何时进入闭环

以下任一情况触发修复闭环：

- `menxia` 裁决为 `封驳`
- 汇总结论中存在 `CRITICAL` / `HIGH` 级问题
- `integrating` 阶段发现多司产出存在核心冲突
- 高风险专项审查给出 `FAIL` 或触及核心目标的 `WARN`

以下情况不进入闭环，直接结束本轮：

- 所有有效结论均为 `PASS`
- 仅剩 `LOW` / `INFO` 级建议，且不影响当前流转
- 所有审查均为 `SKIP`，但当前任务目标只是快速内部讨论

## 2. 闭环角色分工

### `menxia`

- 负责做最终裁决
- 决定是：
  - 直接封驳回修
  - 带风险准奏
  - 启动修复后重审
  - 达到轮次上限后终止

### `shangshu`

- 负责把需修复的问题拆成修订任务包
- 只派给真正受影响的司
- 追踪本轮 `review_round`
- 回收修订结果并提交 `menxia`

### 11 司

- 只修自己负责的问题
- 必须输出“修了什么 / 没法自动修什么 / 建议是否重审”
- 不得借修复之名改写无关模块

## 3. 轮次规则

- 默认 `review_round = 1`
- 每完成一次“修复 -> 回收 -> 重审”，轮次 `+1`
- 默认 `max_review_rounds = 3`

### 终止条件

任一满足即可终止：

1. `menxia` 给出可接受结论（`准奏` 或明确终止）
2. 已达到 `max_review_rounds`
3. 剩余问题均为需人工裁定的争议项
4. 本轮无实质修复执行，只剩延迟处理项

## 4. 选择性重审原则

默认不全量重审，只重审受修复影响的角色。

### 典型映射

- 文案、术语、参数缺失修复
  - 复核：`content-review`
- 核心流程、引导、交互路径修复
  - 复核：`experience-review`
- 公式、成长曲线、概率、经济闭环修复
  - 复核：`balance-audit` / `economy-audit` / `monetization-audit`
- 系统边界、依赖、状态机修复
  - 复核：`system-audit`

### 何时全量重审

只有在以下情况才建议全量重审：

- 修复已改变原始旨意中的核心假设
- 牵涉多个系统的级联影响无法清晰切分
- 上一轮修复后仍出现同类高风险问题

## 5. 修复任务包模板

```markdown
# 修复任务包

## 当前轮次
- review_round: 2 / 3

## 来源裁决
- 封驳 / FAIL / 高风险 WARN

## 本轮需修复问题
- MX-001: [问题摘要]
- BA-003: [问题摘要]

## 指派对象
- `economy`: [要修什么]
- `balancing`: [要修什么]

## 修复边界
- 只允许修改与上述问题直接相关的段落 / 表格 / 规则
- 禁止顺手扩写无关方案

## 回收要求
- 修复说明
- 延迟处理项
- 是否建议重审
```

## 6. 修复回收包模板

```markdown
# 修复回收包

## 本轮修复摘要
- 修复了什么

## 修复明细
- [location] before -> after

## 延迟处理项
- 哪些问题需要人工裁定

## 重审建议
- should_re_review: true / false
- 建议重审角色：
- 原因：
```

## 7. 争议项处理

以下冲突允许标记为 `DISPUTED`，不得强行伪装成“已修好”：

- 乐趣 vs 平衡
- 完整性 vs 简洁性
- 创新性 vs 可行性
- 商业目标 vs 玩家公平感

对争议项的处理规则：

- 若影响核心目标：门下省按保守原则 `封驳`
- 若不影响核心目标但影响体验：可 `准奏`，但必须附风险说明
- 若需要人类拍板：在达到轮次上限前可继续一轮；超过上限则终止并上呈待决项

## 8. 推荐终局格式

闭环结束后，`menxia` 或 `shangshu` 至少说明：

- 当前轮次 / 最大轮次
- 已修复的问题数
- 仍未解决的问题数
- 是否存在争议项
- 最终建议：继续推进 / 局部推进 / 暂停等待人工裁定
