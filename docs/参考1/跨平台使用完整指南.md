### 方案 C：开发者集成（30分钟）

**Python + OpenAI API：**
```python
from openai import OpenAI

with open('游戏开发文档助手-通用AI系统提示词.md', 'r') as f:
    system_prompt = f.read()

client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4",
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": "设计每日签到系统文档"}
    ]
)
print(response.choices[0].message.content)
```

**Python + Google Gemini API：**
```python
import google.generativeai as genai

with open('游戏开发文档助手-通用AI系统提示词.md', 'r') as f:
    system_prompt = f.read()

model = genai.GenerativeModel(
    model_name='gemini-pro',
    system_instruction=system_prompt
)
response = model.generate_content("设计每日签到系统文档")
print(response.text)
```

### 完整版 vs 精简版提示词

| 特性 | 完整版 | 精简版 |
|------|--------|--------|
| 字数 | ~15,000 | ~1,000 |
| Token | ~10,000 | ~700 |
| 详细程度 | 极详细 | 精要 |
| 示例数量 | 多 | 少 |
| 适用场景 | 系统配置、知识库 | 临时使用、API |

**建议：**
- **系统配置时**：用完整版（NotebookLM、自定义 GPT、AI Studio）
- **API 调用时**：用精简版（节省 token）
- **临时使用时**：用精简版（快速粘贴）
