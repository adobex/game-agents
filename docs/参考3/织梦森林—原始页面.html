为什么要这么做？

三七互娱的买量节奏高度贴合"节日场景型"消费品逻辑——种花赛道的核心付费转化窗口不是日常，而是情人节、春节、妇女节等"送花刚需"场景。因此2月5日开投并非偶然，而是精准卡位情人节（2.14）前9天的蓄水期，给系统足够的学习时间完成模型跑通，在2.14当天实现投放效率峰值。后续妇女节（3.8）则作为第二波爆发窗口，形成"双峰"投放曲线。

03 / Creative Strategy

素材创意拆解

DataEye ADX监测显示，《织梦森林》的素材创意围绕三大核心套路展开，形成"情感吸引→萌趣留存→攀比转化"的完整漏斗。

01

乙女向送花剧情

高吸量 情感钩子

美型男角色（多种人设：霸总、温柔学长、病娇少年）向女主角送花的动画剧情。核心钩子是"他为你种了一座花园"的情感叙事。

占比 ~40% ・ CTR 偏高 ・ 精准女性用户
02

萌宠养成陪伴

泛用户吸量 高留存

以萌宠角色为主角，展示养成+陪伴+种花的温馨日常。画面以暖色调为主，强调"治愈系"属性，降低进入门槛。

占比 ~30% ・ 留存偏高 ・ 泛休闲用户
03

社交对比+时装炫耀

高转化 付费驱动

通过"闺蜜花园对比"的动画剧情，制造社交攀比张力——"她的花园比你的美10倍"。重点展示精美时装、稀有花种，刺激付费欲望。

占比 ~30% ・ 付费率偏高 ・ 中重度用户

素材效能矩阵：吸量 vs 留存 vs 转化

素材迭代逻辑

平衡"高吸量"与"高留存"的方法论

A 前3秒钩子分离：乙女剧情素材负责"停留"（Thumb-stop），种花玩法画面负责"说服"（Convince）。前3秒用情感画面拉停，3~10秒切入真实玩法界面，避免"看广告来的用户与实际玩法不符"导致次留崩塌。
B AB组素材并行投放：同一批投放中，乙女剧情素材（高CTR低留存）与萌宠养成素材（中CTR高留存）按约4:3比例混投，用前者拉量、后者稳质。
C 节点限定素材冲击波：情人节/妇女节期间集中上线"社交对比+时装"素材，利用节日消费心理拉高ARPU。平日则回归种花+养成的日常素材。
D 素材衰减周期管理：单条素材生命周期约5-7天，通过每日上新50~100组素材保持投放池活性。衰减素材中CTR仍在线但CVR下降的，调整落地页后重新投放。

04 / Bidding & Logic

出价与投放技术

基于三七互娱的历史投放风格与种花赛道的商业特性，推测《织梦森林》采用分阶段出价策略，在不同生命周期使用不同的优化目标。

出价模式推测

出价策略逻辑

PHASE 1 — 预热期 (Lowest Cost)

目标：最大化安装量，快速积累样本。以"激活"为优化目标，不设出价上限（Lowest Cost/Auto Bid）。原因：新产品冷启动需要大量转化数据喂给系统模型，此阶段"跑量>ROI"。

PHASE 2 — 爆发期 (tCPA → tROAS)

目标：拉高付费用户占比。切换至"付费"为优化事件，设定tCPA（目标付费成本）。情人节/妇女节节点进一步切换为tROAS（目标回收），利用节日ARPU飙升拉高整体ROI。

PHASE 3 — 稳定期 (混合出价)

目标：平衡规模与效率。核心计划用tROAS保ROI，补量计划用oCPM Lowest Cost保规模。同时开启"自动版位"扩展至SDK广告位，降低边际成本。

关键判断

三七互娱作为国内买量实力最强的厂商之一（2025年小游戏发行商入榜数第一、7款入Top100），其核心优势在于投放系统化与数据回传基建。在微信生态中，三七与腾讯广告的深度合作使其享有更精准的用户标签回传（如付费金额、LTV预测），这意味着其tROAS出价模型可以比中小团队更早收敛到目标值。这是"资源型竞争"的典型体现——同一条赛道，大厂跑同样的素材，系统模型收敛更快、买量成本更低。

「侧翼进攻」的三大创新

1 二次元审美降维打击

当所有种花竞品都在走"古风/写实"路线时，《织梦森林》用二次元乙女画风切入，瞄准的是"乙游女性用户"这一高付费意愿群体。这些用户在《恋与深空》《时空中的绘旅人》中已验证了为"美型角色"付费的习惯，种花+换装不过是另一个消费场景。

2 抽卡机制注入种花赛道

竞品的商业化以"数值付费"（加速/体力/扩地）为主，《织梦森林》引入了花种稀有度+抽卡体系，这是将二次元游戏的成熟付费模型（角色/卡池/保底）嫁接到种花赛道的首次尝试，理论上可以显著延长LTV。

3 换装系统作为第二付费引擎

时装+花种主题绑定形成"双重消费触点"——节日活动中限定花种与限定时装同步上线，用户既要抽花又要买装，ARPU上限显著高于纯种花产品。

品牌营销 × 效果广告的协同

微信红包封面 → 社交破圈

春节期间发放品牌红包封面，借微信社交裂变触达非游戏用户。红包封面本身就是一次免费的品牌曝光，接收者点击封面即可跳转小游戏，实现"品牌曝光→用户获取"的零成本闭环。

短视频KOL/KOC → 内容种草

在抖音、快手投放种花类生活博主的种草视频，以"真花开箱"作为内容钩子。这类UGC内容既是效果广告素材，也是品牌信任背书——"真的能收到花"。

节日场景绑定 → O2O送花

情人节、妇女节期间推出"种花送她真花"的主题活动，将游戏行为与现实礼赠场景深度绑定。这不仅是营销活动，更是商业模式本身——O2O履约即产品核心卖点。

好友邀请+花束赠送 → 社交裂变

微信生态内的好友邀请返利、花束互赠机制，利用女性用户群体中天然的社交传播链（闺蜜/同事/家人）实现低成本获客。

赛道竞争格局演进

种花赛道正从"黑马驱动"阶段进入"大厂入局"的结构化竞争阶段

CONCLUSION

总结与前瞻

WHY THIS MATTERS

《织梦森林》的意义不在于它的畅销榜排名，而在于它代表了大厂系统化能力在新兴品类的快速复制。三七互娱用"种花得真花"这一已验证模型 + 二次元乙女审美差异化 + 换装/抽卡双引擎商业化，在一个月内完成了从0到Top47的冷启动，验证了大厂在种花赛道的可行性。

KEY RISKS

赛道买量成本正在快速攀升，素材同质化加剧。"种花送真花"从创新卖点变为行业标配后，用户心智红利消退。《织梦森林》能否长期站稳Top50，取决于内容迭代频率（新花种/新角色/新剧情）和O2O履约体验（真花质量、物流速度、用户复购）。

WHAT'S NEXT

预判后续动作：① 母亲节（5月）将成为下一个投放高峰窗口；② 可能引入明星/虚拟偶像代言强化品牌层；③ 换装系统有望进化为"花艺时装秀"社交玩法，增加UGC传播性；④ 与鲜花品牌的跨界联名（如花点时间、野兽派）可能是提升O2O信任度的路径。
